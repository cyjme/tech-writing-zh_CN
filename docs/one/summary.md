# 《技术写作一》总结

《技术写作一》涵盖了以下技术写作的基本课程：

* 始终如一地使用术语。
* 避免模​​棱两可的代词。
* 主动语态优于被动语态。
* 选择具体的动词而不是模糊的动词。
* 将每个句子集中在一个想法上。
* 将一些长句子转换为列表。
* 删除不需要的词。
* 当排序很重要时使用编号列表，当排序无关时使用项目符号列表。
* 保持列表项平行。
* 用祈使词开始编号的列表项。
* 适当地介绍列表和表格。
* 创建伟大的开篇句子，建立一个段落的中心点。
* 将每个段落集中在一个主题上。
* 确定你的读者需要学习什么。
* 使文档适合您的读者。
* 在文档的开头建立文档的关键点。
